# Set-UsingStatementOrder

## SYNOPSIS

Sort using statements in the current file.

## SYNTAX

```powershell
Set-UsingStatementOrder
```

## DESCRIPTION

The Set-UsingStatementOrder function will sort using statements by type (e.g. Assembly \> Module \> Namespace) and then alphabetically.

## EXAMPLES

### -------------------------- EXAMPLE 1 --------------------------

```powershell
Set-UsingStatementOrder
```

Sort using statements in the current file.
